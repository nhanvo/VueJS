NOTE:<br/>
    
    server:<br/>
        - create file .evn copy from .evn.example<br/>
        - Generate API_KEY: adonis key:generate<br/>
        - run server: adonis serve --dev<br/>
    
    client:<br/>
        - install yarn: brew install yarn<br/>
        - run client: yarn serve<br/>
